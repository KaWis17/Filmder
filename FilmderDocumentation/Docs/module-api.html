(static, constant) fetchMovies

This function returns json which contains one page of films from API (19 films without possability to get more than 19 at once). Function returns filtered and sorted films due to options in variable {params}. If {params} is undefined then function returns films from the list of trending movies of the day. Check this for more filters and sort options https://developer.themoviedb.org/reference/movie-details.
